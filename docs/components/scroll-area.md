## API

| Prop | Description | Type | Default |
|------|-------------|------|---------|
| height | Fixed height | `string \| number` | - |
| maxHeight | Maximum height | `string \| number` | - |
| shadow | Whether to show brutal shadow | `boolean` | `false` |
