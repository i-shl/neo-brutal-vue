# Step

Single step item. Must be placed inside NeoSteps. See [Steps](/components/steps) for usage.

## API

| Prop | Description | Type |
|------|-------------|------|
| title | Title | `string` |
| description | Description | `string` |
| icon | Icon | `string` |
| status | Status | `'wait' \| 'process' \| 'finish' \| 'error' \| 'success'` |
