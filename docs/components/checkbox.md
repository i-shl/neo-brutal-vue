## API

| Prop | Description | Type |
|------|-------------|------|
| modelValue | Bound value | `boolean \| array` |
| label | Option value (for group) | `string \| number \| boolean` |
| disabled | Disabled | `boolean` |
| size | Size | `ComponentSize` |
