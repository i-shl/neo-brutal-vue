# Link

Text link component with Neobrutalism style.

## Basic Usage

<demo vue="../demos/link-basic.vue" />

## API

| Prop | Description | Type | Default |
|------|-------------|------|---------|
| href | URL | `string` | - |
| underline | Show underline | `boolean` | `true` |
| target | Target, e.g. `_blank` | `string` | - |
