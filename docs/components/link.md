# Link

Text link component with Neobrutalism style.

## Basic Usage

```vue
<template>
  <NeoLink href="https://github.com">Default Link</NeoLink>
</template>
```

## API

| Prop | Description | Type |
|------|-------------|------|
| href | URL | `string` |
| target | Target, e.g. `_blank` | `string` |
