## API

| Prop | Description | Type |
|------|-------------|------|
| src | Image URL | `string` |
| icon | Icon (emoji or class name) | `string` |
| size | Size | `ComponentSize \| number` |
| shape | Shape | `'circle' \| 'square'` |
| alt | Image alt | `string` |

Slots: default (text or custom content).
