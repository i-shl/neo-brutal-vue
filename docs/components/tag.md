## API

| Prop | Description | Type |
|------|-------------|------|
| type | Type | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'default'` |
| size | Size | `'xs' \| 'sm' \| 'md' \| 'lg'` |
| effect | Effect | `'light' \| 'dark' \| 'plain'` |
| closable | Whether closable | `boolean` |
