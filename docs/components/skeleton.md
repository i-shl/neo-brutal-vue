# Skeleton

Placeholder skeleton for loading content.

## Basic Usage

```vue
<template>
  <NeoSkeleton />
</template>
```

## Multiple Rows & Custom

```vue
<template>
  <NeoSkeleton :rows="4" />
</template>
```

## API

| Prop | Description | Type |
|------|-------------|------|
| rows | Number of paragraph rows | `number` |

Slots: default (content to show when loading completes).
