## API

| Prop | Description | Type |
|------|-------------|------|
| modelValue | Bound value | `boolean \| string \| number` |
| disabled | Disabled | `boolean` |
| size | Size | `ComponentSize` |
| activeText / inactiveText | Switch text | `string` |
