## API

| 属性 | 说明 | 类型 |
|------|------|------|
| modelValue | 绑定值 | `boolean \| string \| number` |
| disabled | 禁用 | `boolean` |
| size | 尺寸 | `ComponentSize` |
| type | 选中时的颜色类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'default'` |
| activeText / inactiveText | 开关文字 | `string` |
