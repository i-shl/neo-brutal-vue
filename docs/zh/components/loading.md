## API

| 属性 | 说明 | 类型 |
|------|------|------|
| text | 加载文字 | `string` |
| fullscreen | 是否全屏 | `boolean` |
| lock | 是否锁定滚动 | `boolean` |
| background | 遮罩背景色 | `string` |
