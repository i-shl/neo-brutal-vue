## API

| 属性 | 说明 | 类型 |
|------|------|------|
| loading | 是否显示加载 | `boolean` |
| text | 加载文字 | `string` |
| size | 加载图标尺寸 | `'small' \| 'default' \| 'large'` |
| fullscreen | 是否全屏 | `boolean` |
| type | 加载图标颜色类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'default'` |
