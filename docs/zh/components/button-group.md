# ButtonGroup 按钮组

将多个按钮组合在一起，消除重叠边框，使其看起来像一个连贯的组件。

## 基础用法

使用 `<NeoButtonGroup>` 将多个 `<NeoButton>` 包裹起来，即可合并它们相邻的粗黑边框，形成新布鲁塔利主义中非常经典的“拼图模块”视觉效果。

## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| - | 无 Props，作为子按钮的 CSS 布局与边框容器。 | - | - |
