## API

| 属性 | 说明 | 类型 |
|------|------|------|
| modelValue | 绑定值 | `boolean \| array` |
| label | 选项值（用于组） | `string \| number \| boolean` |
| disabled | 禁用 | `boolean` |
| size | 尺寸 | `ComponentSize` |
| type | 选中时的颜色类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'default'` |
