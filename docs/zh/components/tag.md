## API

| 属性 | 说明 | 类型 |
|------|------|------|
| type | 类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'default'` |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg'` |
| effect | 效果 | `'light' \| 'dark' \| 'plain'` |
| closable | 是否可关闭 | `boolean` |
