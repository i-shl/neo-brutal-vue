## API

| 属性 | 说明 | 类型 |
|------|------|------|
| description | 描述文字 | `string` |
| image | 自定义图片地址 | `string` |
| showImage | 是否显示图片 | `boolean` |
| type | 占位框颜色类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'default'` |

插槽：默认（自定义内容）、`image`、`footer`。
