## API

| 属性 | 说明 | 类型 |
|------|------|------|
| value | 显示值（数字或文字） | `string \| number` |
| max | 最大值，超过显示 max+ | `number` |
| isDot | 是否显示为圆点 | `boolean` |
| hidden | 是否隐藏 | `boolean` |
| type | 类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info'` |
