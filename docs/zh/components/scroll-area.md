## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| height | 固定高度 | `string \| number` | - |
| maxHeight | 最大高度 | `string \| number` | - |
| shadow | 是否显示粗黑阴影 | `boolean` | `false` |
