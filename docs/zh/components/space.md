## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| size | 间距大小 | `'sm' \| 'md' \| 'lg' \| number` | `'md'` |
| vertical | 是否垂直排列 | `boolean` | `false` |
